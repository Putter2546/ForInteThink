# ระบบจราจรอัจฉริยะ - Smart Traffic System

ระบบแสดงข้อมูลการจราจรและบริการที่เกี่ยวข้องแบบเรียลไทม์ พัฒนาด้วย HTML, JavaScript และ Tailwind CSS

## ฟีเจอร์หลัก

1. **แสดงสภาพการจราจรแบบเรียลไทม์** - แสดงสถานะการจราจรบนเส้นทางต่างๆ พร้อมความเร็วและเวลาเดินทาง
2. **พยากรณ์ช่วงเวลารถติด** - พยากรณ์ความหนาแน่นของการจราจรในแต่ละช่วงเวลา
3. **แนะนำเส้นทางที่เหมาะสม** - แนะนำเส้นทางที่ดีที่สุดจากจุดเริ่มต้นไปยังจุดหมายปลายทาง
4. **ระบบที่จอดรถ** - แสดงข้อมูลที่จอดรถที่มีอยู่ พร้อมจำนวนที่ว่างและราคา
5. **ระบบรถรับส่ง (Shuttle / Bus)** - แสดงข้อมูลเส้นทางรถรับส่งและเวลามาถึงครั้งถัดไป
6. **แจ้งเตือนและประกาศ** - แสดงประกาศและแจ้งเตือนต่างๆ ที่เกี่ยวข้อง
7. **ระบบรายงานจากผู้ใช้** - ระบบให้ผู้ใช้สามารถรายงานปัญหาต่างๆ ได้

## การใช้งาน

### รันแบบ Local

1. เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์
2. หรือใช้ Live Server ใน VS Code

### Deploy บน Vercel

1. ติดตั้ง Vercel CLI (ถ้ายังไม่มี):
   ```bash
   npm i -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```

3. หรือ Deploy ผ่าน GitHub:
   - Push โค้ดไปยัง GitHub repository
   - ไปที่ [Vercel](https://vercel.com)
   - Import project จาก GitHub
   - Vercel จะ detect และ deploy อัตโนมัติ

## เทคโนโลยีที่ใช้

- **HTML5** - โครงสร้างหน้าเว็บ
- **JavaScript (Vanilla)** - ฟังก์ชันการทำงาน
- **Tailwind CSS** - การออกแบบ UI (ผ่าน CDN)
- **Font Awesome** - ไอคอน
- **Google Fonts (Kanit)** - ฟอนต์ภาษาไทย

## หมายเหตุ

- ระบบนี้ใช้ข้อมูลจำลอง (Mock Data) สำหรับการแสดงผล
- ไม่มี Backend Server - ข้อมูลจะถูกเก็บใน LocalStorage ของเบราว์เซอร์
- รายงานจากผู้ใช้จะถูกเก็บใน LocalStorage ของแต่ละเครื่อง
